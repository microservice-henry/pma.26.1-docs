# Product API

!!! warning "Em desenvolvimento"
    O Product Service será implementado em uma próxima entrega. Esta seção será atualizada com endpoints, modelo de dados e instruções de execução.

## Planejado

O Product Service será responsável por:

- Cadastro e consulta de produtos (nome, descrição, preço)
- Exposição do endpoint `GET /products/{id}` consumido pela Order API

Durante o desenvolvimento atual da Order API, o Product Service é **simulado por WireMock** — veja [Como Rodar](../order/run.md) para detalhes dos mocks.
